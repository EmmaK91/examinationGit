
# Todo Lite – Individuell examination

En enkel Todo-applikation där användare kan:

- Lägga till uppgifter
- Markera uppgifter som klara eller oklara
- Ta bort uppgifter
- Validera att uppgifter inte är tomma

## Installation

1. Klona repot:
git clone https://github.com/EmmaK91/examinationGit
Öppna index.html i valfri webbläsare.


Funktioner
Lägg till uppgift: Skriv i inputfältet och klicka "Lägg till".
Markera klar/oklar: Klicka på "Klar"/"Ångra".
Ta bort uppgift: Klicka på "Ta bort".
Felhantering: En tom uppgift kan inte läggas till (visar ett felmeddelande).

LocalStorage: Sparar uppgifterna lokalt mellan sidladdningar.

Struktur
index.html – HTML-struktur och UI.
style.css – Grundläggande styling.
app.js – Funktionalitet: add, remove, toggle, render, localStorage, felmeddelande.

Test
Testat manuellt genom att lägga till, ta bort och markera uppgifter.
Felmeddelande visas korrekt när input är tomt.
